# Assumptions & Scope

## In Scope
- Customer login and authentication (password + MFA)
- Web application and API
- Cloud infrastructure (networking, IAM)
- Customer data storage (PII + financial data)
- Logging and monitoring
- Third-party integrations (email/SaaS)

## Out of Scope
- Physical security and on-prem data centers
- Full code review / secure SDLC implementation details
- Penetration testing results (not performed)
- Mobile app-specific threats (unless explicitly added)

## Assumptions
- Cloud-hosted environment (AWS or Azure equivalent)
- Standard enterprise identity controls are partially implemented
- App supports user transactions and account servicing
